# Architecture of youkyouk

- **Backend**: Node.js with Express.js (Medusa-like structure)  
- **Frontend**: Next.js with React  
- **Database**: To be decided (PostgreSQL recommended)  
- **Authentication**: JWT-based  
- **Modular design**: Plugins system for features  
- **Multi-tenancy**: Supports multiple markets and vendors  

This architecture allows flexibility and scalability.
