Installation Requirements

Please make sure your server or local environment meets the following requirements before starting the setup:

  • Node.js >= 18
  • npm >= 9
  • PostgreSQL >= 14
  • Operating System: Linux / macOS / Windows with terminal access

Installation Steps

  1. Download and extract the package youkyouk-package.zip provided with your purchase.
  2. Open a terminal and navigate to the project folder:
    cd youkyouk
  3. Run the installer script:
    ./install.sh
  4. Follow the instructions on screen to complete the setup.
